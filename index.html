Web projects

RatesPro Currency Converter

PHP, HTML, CSS, JavaScript

This project is a professional currency conversion web application that provides real-time exchange rates using the ExchangeRate-API. It features a clean, responsive interface where users can select currencies, input amounts, and instantly see conversion results. The application demonstrates secure API key handling through backend PHP logic, includes comprehensive error handling, and provides additional functionality like currency swapping and historical rate display, showcasing full-stack development capabilities.

HealthTrack BMI Calculator

PHP, HTML, CSS, JavaScript

This project is an interactive Body Mass Index calculator that allows users to calculate their BMI using both metric and imperial measurement systems. It provides immediate visual feedback by displaying results on a color-coded health scale and offers personalized health interpretations. The application features a professional healthcare-themed design, responsive layout, and client-side form validation, demonstrating strong front-end development skills while maintaining PHP backend readiness for data processing and storage.

TaskMaster To-Do List

HTML, CSS, JavaScript

This project is a clean, responsive to-do list application that allows users to add, edit, delete, and mark tasks as complete. It features filtering options to view all tasks, active tasks, or completed tasks, along with statistics tracking. The application demonstrates core JavaScript concepts like DOM manipulation, event handling, and local data management, making it an excellent learning project for understanding how interactive web applications work.
